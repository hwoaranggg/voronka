# Бот-продажник мануала по чаттингу

Aiogram 3. Воронка: диплинк-трафик → квиз → персональный оффер → оплата (карта/крипта) → автовыдача одноразовой инвайт-ссылки в закрытый канал. Плюс админка со статой и рассылкой.

## Структура

| Файл | За что отвечает |
|------|-----------------|
| `bot.py` | Хендлеры, квиз, оплата, админка, фоновый поллер |
| `texts.py` | **Все тексты и квиз** — правишь здесь, логику не трогаешь |
| `config.py` | Настройки из `.env` |
| `db.py` | SQLite (пользователи, оплаты, стата) |
| `payments.py` | CryptoBot + YuKassa |

## Установка

```bash
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env             # заполни значения
python bot.py
```

## Что нужно подготовить

1. **Токен бота** — у @BotFather → `BOT_TOKEN`.
2. **ID канала** — закрытый канал, добавь бота админом с правом приглашать. `CHANNEL_ID` формата `-100...`.
3. **Свой Telegram ID** → `ADMIN_IDS` (узнать у @userinfobot).
4. **Оплата:**
   - Крипта: токен в @CryptoBot → Crypto Pay → Create App → `CRYPTOBOT_TOKEN`.
   - Карта: `YUKASSA_SHOP_ID` + `YUKASSA_SECRET_KEY` из ЛК ЮKassa.

## Про оплату картой в нише 18+

ЮKassa нередко **не проходит модерацию для 18+**. Если откажут — переключись на Tribute:
поставь `CARD_PROVIDER=tribute` и `TRIBUTE_LINK=<ссылка на товар в @tribute>`.
Тогда бот отдаёт ссылку, а доступ в канал выдаёт сама Tribute (настраивается в её интерфейсе).
Крипта (CryptoBot) для этой ниши работает стабильно — её можно оставить основным способом.

## Диплинки для трафика

Метка источника зашивается в ссылку:
`https://t.me/ТВОЙ_БОТ?start=tiktok_video1`
Метка (`tiktok_video1`) сохраняется и видна в админке `📊 Статистика` → разбивка по источникам.
Делай уникальную метку под каждый ролик/площадку — увидишь, что реально конвертит.

## Админка

`/admin` (только для `ADMIN_IDS`):
- 📊 Статистика — юзеры, оплаты, конверсия, выручка, источники
- 📣 Рассылка — пересылает твоё сообщение всем (текст/фото/видео), с защитой от лимитов

## Как редактировать квиз

В `texts.py`, список `QUIZ`. Добавляй/убирай вопросы — бот сам подстроится.
Персональную подводку к офферу правишь в `build_pitch()`.

## Деплой

Railway / любой VPS. Поллинг, вебхук не нужен — публичный домен не требуется.
На Railway: добавь переменные из `.env` в Variables, старт-команда `python bot.py`.
